# Security Policy

## Supported Versions

The latest version of Elm Land will receive security
patches. At the time of writing, here are the latest
versions:

| Version    | Supported          |
| ---------- | ------------------ |
| 0.20.0     | :white_check_mark: |
| < 0.20.0   | :x:                |
